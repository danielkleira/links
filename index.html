Comandos

Checar versão e instalação

git version

Abrir pasta com git bash

code .

Iniciando um repositório local

git init

Verificar arquivos locais

ls -a

Adicioanar ao rastreio do Git

git add .

Configurar nome

git config --global user.name seuNome

Configurar e-mail

git config --global user.email seuEmail

Commitando

git commit -m "mensagem"

Gerando uma chave SSH

ssh-keygen -t ed25519 -C "seuEmail@dominio.com"

Pegar a chave SSH no CTRL + V

clip < ~/.ssh/id_ed25519.pub

Associar repositório local e online

git remote add origin “link copiado”

Associar branchs

git branch -M main

Enviando seus arquivos para o repositorio online

git push -u origin main

Clonar repositorio

git clone
